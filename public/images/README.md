# STAGE Images

## How to Add Images

### 1. Randeep Hooda Image (Brand Ambassador)
Place Randeep Hooda's image in this folder with the name:
```
randeep-hooda.jpg
```

**Image specifications:**
- Format: JPG, PNG, or WEBP
- Aspect Ratio: 3:4 (portrait)
- Recommended size: 600x800px or higher
- File location: `/public/images/randeep-hooda.jpg`

### 2. STAGE Logo
- Already added: `stage-logo.svg`
- Update this file if you have the actual STAGE logo from the STAGE app

## Where to Get These Images

### From STAGE App:
1. Navigate to the STAGE app directory
2. Look for assets/images or public/images folder
3. Copy the following files:
   - Brand ambassador photo (rename to `randeep-hooda.jpg`)
   - STAGE logo (replace `stage-logo.svg`)

### Manual Upload:
1. Get the official images from STAGE marketing/branding team
2. Copy them to this folder: `/public/images/`
3. Ensure file names match exactly:
   - `randeep-hooda.jpg` (brand ambassador)
   - `stage-logo.svg` or `stage-logo.png` (logo)

## After Adding Images

The landing page will automatically use these images. If images are not found, it will show placeholders.

Refresh your browser (Ctrl+Shift+R or Cmd+Shift+R) to see the changes.
